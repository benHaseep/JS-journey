```js
let askForEmail = prompt("Enter Your Email..");

let email = "benhaseep@outlook.com";

  

console.log(askForEmail.toLowerCase().trim == email);
// false -> prompt: sdkfjsdf
// true -> prompt: benHaseep@outlook.com

// toLowerCase()
// toUpperCase()
// trim() -> delete the spaces
```

convert the string to lower case and upper case.
```js
let name = "Ali"

console.log(name.toLowerCase()); // ali
console.log(name.toUpperCase()); // ALI

```
