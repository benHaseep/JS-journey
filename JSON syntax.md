- Data added inside curly braces
- Data added with `key : value`
- Key should be string wrapped in double quotes.
- Data separated by coma
- square brackets `[ ]` for arrays
- Curly braces `{ }` for objects

**Available Data Types:**
- String
- Number
- Object 
- Array
- Boolean values
- null

```json
{
	"string": "Haseep",
	"num": 25,
	"object": {
		"EG": "Zag",
		"KSA": "Riyadh",
		},
	"array": ["HTML", "CSS", "JS"],
	"boolean": true,
	"null": null,
	}
```
