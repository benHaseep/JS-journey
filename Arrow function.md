> Arrow function is an expression type of functions. 
> Introduced in ECMAScript 2015 (ES6)

the basic expression function:
```js
let x = function () {
	console.log("arrow");
}

x();  // arrow
```

the arrow function:
```js
let x = () => {
	console.log("arrow");
}

x(); // arrow
```

```js
let x = () => console.log("arrow");
x(); // arrow
```
